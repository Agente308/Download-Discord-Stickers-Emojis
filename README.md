# Download-Discord-Stickers-Emojis
Aplicación en Python con interfaz gráfica que permite descargar stickers y emojis de Discord usando solo su ID. Los stickers se guardan en 160x160 px y los emojis en 48x48 px. La carpeta de guardado es configurable y se almacena automáticamente. Incluye funciones para copiar la URL generada y abrirla en el navegador.
